# Анкета для подбора гида во Владивостоке

Интерактивная анкета для подбора индивидуального гида во Владивостоке. Проект помогает туристам найти идеального гида, подходящего под их интересы, предпочтения и бюджет.

## Особенности

- **Интерактивная анкета**: 4 раздела с вопросами о путешествии, интересах, пожеланиях к гиду и бюджете
- **4 уникальных гида**: Каждый гид имеет свою специализацию и уникальные особенности
- **Модальное окно**: Результаты отображаются в красивом модальном окне
- **Адаптивный дизайн**: Подходит для просмотра на различных устройствах
- **Фотографии гидов**: Каждый гид представлен с фотографией

## Гиды

### 1. Морозов Максим Петрович (Гид-историк)
- **Специализация**: Глубокие исторические экскурсии, форты, архитектура, военно-морская история
- **Формат**: Пешеходный или на минивэне
- **Особенность**: Показывает архивные фото на планшете для сравнения "было-стало"

### 2. Паровозов Аркадий Аркадьевич (Гид-блогер)
- **Специализация**: Нестандартные маршруты, фото-точки, современная культура
- **Формат**: Пешеходный или на автомобиле
- **Особенность**: Фотосопровождение в подарок, советы по обработке снимков

### 3. Леворгов Андрей Михайлович (Гид-экстремал)
- **Специализация**: Активный отдых, походы, дайвинг, внедорожные выезды
- **Формат**: На внедорожнике или катере
- **Особенность**: Предоставляет снаряжение и страховку

### 4. Соловьёва Мария Олеговна (Гид-аниматор)
- **Специализация**: Семейные экскурсии, гастрономические туры, квесты для детей
- **Формат**: Пешеходный или комбинированный
- **Особенность**: Интерактивные элементы и детские игры

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Адаптивный дизайн

## Как использовать

1. Заполните анкету, ответив на вопросы о ваших предпочтениях
2. Нажмите "Подобрать гида!"
3. В модальном окне появится информация о подобранном гиде
4. Вы можете закрыть окно или заполнить анкету заново

## Запуск

Проект можно запустить, просто открыв файл `index.html` в любом современном браузере.
